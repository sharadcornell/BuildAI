How the 7 pages work

Each page opens with its own full-bleed color block: Home = red, Programme = ink/black, Curriculum = white with red rails, Colleges = yellow, Students = red, Mentors = ink, About = white. Numbered like a manual ("03 — Curriculum") so the whole site reads as one Swiss poster series.

Signature devices

  • 3px ink borders, stacked module rows
  • Outlined + yellow display type
  • Black stat bands on every page
  • Hover = yellow fill (zero JS)

Weight & effort

  • ~0 KB JS, system Helvetica
  • Loads instantly on any phone
  • Lowest effort: extend v5 you have
