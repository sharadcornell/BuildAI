# reference/ — business & design context for Claude Code

Source material for BuildAI, copied in so Claude Code has full context while building.
**None of this is imported by the app** — it won't affect the build or deploy. Safe to delete
before production if you want a leaner repo.

- `BuildAI_Project_Brief.md` — the master project brief (the original CLAUDE.md). Single best
  overview: positioning, numbers, curriculum, brand voice, open issues.
- `current-website/` — the existing HTML the Next.js site is based on. The locked direction is
  `swiss/BuildAI_Swiss_v5_RedBlock_3D_Final.html` (see `swiss/QA Screenshots/` for renders).
  `BuildAI_3D_Website_Inspiration.md` explains the "looks 3D, built 2D" approach.
- `curriculum/` — full 13-week curriculum (`_v2` is current), teaching playbook, the
  MastersUnion benchmark, and `sprints/` (the shorter 1- and 2-weekend course variants).
- `strategy/` — master strategy, positioning one-pager, 90-day implementation plan, financial
  models (`_v2` is current).
- `brochures-and-outreach/` — student brochure, mentor one-pager, TPO outreach email templates.
- `pitch-decks/` — college pitch decks (`_v2` current; plus Swiss and Wayfinding variants).

> The actionable build spec is one level up in `docs/BuildAI_Website_Build_Brief.md`.
